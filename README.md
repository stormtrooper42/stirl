# Stirl

Stirl is a blog system using PHP without framework.

  - Easy
  - Free
  - Fast

### Installation

You need to modify the config file in config/config.php

config.php:

```php
	return array(
		"db_user" => "root",
		"db_pass" => "djason",
		"db_host" => "localhost",
		"db_name" => "blog"
	);
```
